# Warband Nexus - Screenshots

This folder contains screenshots for CurseForge/Wago listing.

## Required Screenshots:

1. **Main Window - Characters Tab** (1920x1080 recommended)
   - Show character list with class colors
   - Display gold amounts
   - Show sorting indicators
   - Favorite stars visible

2. **Items Tab - Search Results** (1920x1080)
   - Show search bar with query
   - Display filtered items
   - Show item quality colors
   - Display location information

3. **Storage Tab - Category View** (1920x1080)
   - Show Warband Bank section
   - Show Personal Bank sections
   - Display item categories
   - Show item counts

4. **PvE Tab - Progress** (1920x1080)
   - Show M+ keystone history
   - Display raid lockouts
   - Show Great Vault indicators

5. **Minimap Button + Tooltip** (800x600)
   - Show minimap button
   - Display tooltip on hover
   - Show quick stats

6. **Notification Examples** (1920x1080)
   - Show loot notification toast
   - Display vault reminder
   - Show stacked notifications

## Tips for Taking Screenshots:

1. Use high resolution (1920x1080 or higher)
2. Clean UI (hide other addons if possible)
3. Use a neutral background (Stormwind/Orgrimmar)
4. Show actual data (not empty states)
5. Capture at different times of day (good lighting)
6. PNG format preferred (better quality)
